ENCOUNTERED PROBLEMS

FPS ISSUE

We encountered a decrease in FPS while modeling our game. We had to make a lot of changes to the game to make it run smoothly. For instance, we had to change the way the environment was generated and rendered.

CAMERAS IN MULTIPLAYER

Using Photon, we had to synchronize players' cameras. In fact, we had to make sure that the cameras of the players were assigned to the correct player. This is a recurring problem.

IMPLEMENTING MINI-GAMES

Mini games are a great way to improve the game. We had to implement a lot of them to make LockedUp more enjoyable. To do so we wanted stay in the same room where the player is. Unfortunately, this was not possible, so we created a separate room for each mini-game.
